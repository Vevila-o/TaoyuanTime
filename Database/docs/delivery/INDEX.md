# 交接交付包目錄（Delivery Index）

本資料夾為本次交接的重點文件集合，設計參考租屋系統的 docs 風格，方便組員一眼看懂並快速上手。

主要內容：

- `QUICK_START.md`：最短跑通步驟（建立虛擬環境、migrate、seed、啟動）
- `QA_CHECKLIST.md`：交接 QA 檢查清單
- `ER_REFERENCE.md`：ER 圖與資料表摘要（含 mermaid 檔）
- `FINAL_DELIVERY_CHECKLIST.md`：此交付包包含哪些檔案與說明

其他參考（不影響主要交付）會收納在 `docs/archived/`，以免干擾主要流程。

請先閱讀 `QUICK_START.md` 再依序執行。
